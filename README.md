## 介绍

【vue全家桶练习项目】仿网易云音乐客户端，目前实现了页面基本样式，播放，歌词，歌单，专辑，歌手，视频，搜索，登录，收藏，查看评论等大部分基础功能。
模仿 网易云 音乐pc端软件界面，采用 flexbox 和 position 布局；虽然是响应式，但主要以 PC 端为主，移动端只做相应适配.
